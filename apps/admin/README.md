# @gigs/admin

Admin dashboard for gigs.ge — user moderation, dispute arbitration, invoice management, and platform oversight.

## Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Data Fetching:** TanStack Query

## Quick Start

```bash
# From monorepo root
pnpm --filter @gigs/admin dev   # Start on port 3002
```

## Build

```bash
pnpm --filter @gigs/admin build
```
